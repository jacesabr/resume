Naive Bayes

Bayes' theorem with feature independence — blazing fast, ideal for text classification and spam filtering.

20 examples across 14 datasets · Individual code snippets · LaTeX equations

Mathematical Foundation

\[P(y|x) = \frac{P(y)\prod_{j=1}^{p}P(x_j|y)}{P(x)}\]

Posterior ∝ Prior × Likelihood (with independence assumption)

\[\hat{y} = \arg\max_c P(C=c)\prod_{j}P(x_j|C=c)\]

Classify by highest posterior probability

\[P(x_j|C=c) = \mathcal{N}(\mu_{jc}, \sigma_{jc}^2)\]

GaussianNB: each feature is Gaussian within each class

\[\text{Laplace smoothing: } P(x_j|c) = \frac{\text{count}(x_j,c) + \alpha}{\text{count}(c) + \alpha|V|}\]

Prevents zero probabilities for unseen features

Glossary of Technical Terms

Bayes' Theorem
P(A|B) = P(B|A)·P(A)/P(B). Update beliefs with evidence.
Prior P(y)
Class probability before seeing features. From training frequencies.
Likelihood P(x|y)
Feature probability given class. What the model learns.
Naive Assumption
Features independent given class. Usually wrong but still works!
MultinomialNB
For counts/frequencies — ideal for TF-IDF text.
GaussianNB
For continuous features — assumes bell curve per class.
Laplace Smoothing (α)
Adds α to counts to prevent zero probabilities.
Zero-Frequency
Unseen feature value → P=0 → posterior=0. Fixed by smoothing.
