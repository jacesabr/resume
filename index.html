How It Works — Code Examples
Memory That Fades

Memories decay over time. Important ones stick. Each recall strengthens stability.

function scoreMemory(mem, now) {
  const age = hoursElapsed(mem.timestamp, now)
  const decay = Math.exp(-age / mem.stability)
  return mem.importance * decay
}
// on recall: mem.stability += 0.1
