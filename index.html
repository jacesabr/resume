What it is: A deeply personalized AI companion named Morrigan — not a generic chatbot, but a character with her own personality, memories, and emotional depth. She remembers past conversations, forms opinions about you over time, progressively reveals vulnerability as trust builds, and generates inner thoughts that shape her responses. The goal: AI that feels like talking to a real person, not a tool.
How It Works
Every message triggers a 6-phase pipeline: detect emotional signals in user input → retrieve relevant memories via embedding similarity → generate 2-3 candidate inner thoughts → score each by relevance, timing, trust gate & novelty → weave the winning thought into her response → stream it back via SSE in real time.
Core Systems
  • Persistent Memory — embeddings + cosine similarity + temporal decay
  • Contradiction Detection — 5-category classifier across conversations
  • Trust Progression — 7 levels gating vulnerability & disclosure
  • Self-Atom Framework — 40 personality atoms across 4 depth tiers
  • Relationship Health — 5-signal model for engagement tracking
