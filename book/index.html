The Book What This Is
"Most LLM books teach you to call an API. This one teaches you to build one from scratch — tokenizer to fine-tuned assistant, every line annotated, every design decision traced to the paper that justified it."

This book is a practitioner's guide to understanding and building large language models — from the mathematics of next-token prediction through pre-training, deployment, and fine-tuning. Each chapter explains one layer of the stack deeply, with annotated code the reader can run in their browser and links to every primary source.

The target reader is someone technical enough to read Python, curious enough to want the real math, but not necessarily a researcher. A senior developer, a data scientist pivoting to LLMs, or a technically-minded builder who wants to go beyond the API.

By the end of this book you will have built a working GPT model from an empty file — tokenizer, attention mechanism, transformer architecture, pre-training loop, and two different fine-tuning strategies — understanding every line of code and every design trade-off along the way.
