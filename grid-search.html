Grid Search

Exhaustive hyperparameter search with cross-validation — guaranteed optimal within the grid.

20 examples across 13 datasets · Individual code snippets · LaTeX equations

Mathematical Foundation

\[\theta^* = \arg\max_{\theta\in\Theta}\frac{1}{K}\sum_k \text{Score}(\theta,\text{fold}_k)\]

Best params = max CV score

\[|\Theta| = \prod_{i=1}^{h}|V_i|\]

Total combos = product of values per param

\[\text{Total fits} = |\Theta| \times K\]

Each combo × K folds

\[\text{RandomizedSearch samples } n \text{ combos randomly}\]

Often finds ~same result in 1/10th time

Glossary of Technical Terms

Hyperparameter
Set BEFORE training — not learned from data (C, max_depth, k).
Parameter Grid
Dict of param→values. {'C':[0.1,1,10], 'gamma':[0.01,0.1]}.
GridSearchCV
Exhaustive search — tries every combination with CV.
RandomizedSearchCV
Random sampling — much faster for large grids.
best_params_
Optimal combination found.
best_score_
Highest CV score achieved.
n_jobs=-1
Parallelize across all CPUs.
Overfitting to CV
Too many combos → finds one that happens to score well on CV folds.

from sklearn.model_selection import GridSearchCV
from sklearn.ensemble import RandomForestClassifier

param_grid = {'svc__C': [0.1,1,10,100], 'svc__gamma': ['scale',0.001,0.01,0.1]}
print(f'Parameter grid:')
for param, values in param_grid.items():
    print(f'  {param}: {values}')

total_combos = 1
for v in param_grid.values():
    total_combos *= len(v)
print(f'\nTotal combinations: {total_combos} × 5 folds = {total_combos*5} model fits')

gs = GridSearchCV(RandomForestClassifier(random_state=42), param_grid, cv=5, n_jobs=-1, return_train_score=True)
gs.fit(X, y)

# --- RESULTS ---
print(f'\n{'='*50}')
print(f'Best CV score: {gs.best_score_:.4f} ({gs.best_score_*100:.1f}%)')
print(f'Best parameters: {gs.best_params_}')

# Show all results ranked
import pandas as pd
results = pd.DataFrame(gs.cv_results_)[['params','mean_test_score','std_test_score','mean_train_score','rank_test_score']]
results = results.sort_values('rank_test_score').head(5)
print(f'\nTop 5 parameter combinations:')
print(results.to_string())

# Check for overfitting: if train >> test, model is too complex
best_train = gs.cv_results_['mean_train_score'][gs.best_index_]
best_test = gs.best_score_
gap = best_train - best_test
print(f'\nOverfit check: train={best_train:.4f}, test={best_test:.4f}, gap={gap:.4f}')
if gap > 0.05:
    print(f'⚠️ Gap > 5% — model may be overfitting. Try simpler parameters.')
else:
    print(f'✓ Gap < 5% — model generalizes well.')
